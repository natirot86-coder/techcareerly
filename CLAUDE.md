# Techcareerly — הוראות לקלוד

## מה הפרויקט הזה?

**Web App היברידי** שמלווה מועמדים במסע בחירת קריירה טכנולוגית.
המערכת משלבת אינטראקציה דיגיטלית עם פגישות פרונטליות עם רכזת.

---

## ארכיטקטורה טכנית

| שכבה | כלי | תפקיד |
|---|---|---|
| Frontend | React / Next.js | ה-Web App של המועמד |
| Database | Supabase | אחסון נתוני המועמד ומשימות |
| Automation | Make.com | לוגיקה עסקית, Webhooks, ניתוח AI |
| CRM | Monday.com | ממשק הרכזת, התראות, סטטוסים |
| AI Co-pilot | Claude API | צ'אט תמיכה למועמד |

---

## מסע המשתמש (6 שלבים)

1. **טרום אינטייק** — SMS/WhatsApp → הורדת האפליקציה → שאלון בסיס
2. **אינטייק חי** — פגישה פרונטלית → רכזת מאשרת במאנדיי → פתיחת מפת הדרכים
3. **חשיפה** — סימולציות מקצועיות → ניתוח AI → פגישה 2
4. **מסלול לימודים** — חקר מוסדות → פגישה 3: נעילת מסלול
5. **לוגיסטיקה** — הכנה פיננסית ומלגות
6. **רישום** — אימות רישום סופי → סטודנט רשום

פגישות עם רכזת (שלבים 2, 3, 4) מסומנות בכתום ב-PRD.

---

## מסכי האפליקציה

- **Login** — מסך הזדהות
- **Dashboard** — מפת הדרכים (מרכז הניווט)
- **Learning** — מרכז למידה ומשימות
- **AI Chat** — צ'אט Co-pilot
- **Squad** — קיר השראה וקהילה
- **Contact** — קשר עם הרכזת

---

## לוגיקת ה-AI (Nudge System)

המערכת מזהה חסמים ומתריעה לרכזת:

- **חוסר פעילות > 72 שעות** → סטטוס "At Risk" במאנדיי
- **כישלון במשימה 3 פעמים** → משימה דחופה לרכזת
- **מילות תסכול בצ'אט** → ניתוח סנטימנט → "Manual Intervention Required"
